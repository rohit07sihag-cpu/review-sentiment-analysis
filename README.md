# Product Review Sentiment Analyzer

## Overview

This project uses a pretrained Hugging Face Transformer model to classify customer reviews as Positive or Negative.

## Tech Stack

- Python
- Google Colab
- Hugging Face Transformers
- PyTorch

## Features

- AI-based sentiment analysis
- Classifies customer reviews
- Displays prediction confidence
- Works without training a custom model

## Results

The model successfully classified both positive and negative reviews with high confidence.

## How to Run

1. Open the notebook in Google Colab.
2. Run all cells.
3. View the predicted sentiment for each review.
